📈 Distribuciones Estadísticas

Describe cómo se comportan los datos de un proceso de medición. La distribución normal (Gauss) es fundamental en metrología.

📐 Características:

• Media (μ): Valor central de las mediciones
• Desviación Estándar (σ): Dispersión de los datos
• 68.3% de datos en μ ± σ
• 95.4% en μ ± 2σ
• 99.7% en μ ± 3σ

f(x) = (1/(σ√(2π))) · e^(-(x-μ)²/(2σ²))

✓ Prueba de Normalidad

Verifica si los datos siguen una distribución normal. Es requisito previo para ANOVA y pruebas t.

🔬 Métodos comunes:

Shapiro-Wilk: Para muestras pequeñas (n < 50)
Kolmogorov-Smirnov: Para muestras grandes
Anderson-Darling: Mayor sensibilidad en colas

📋 Interpretación:

p-valor > 0.05 → Datos normales (H₀ aceptada)
p-valor ≤ 0.05 → Datos no normales

📊 ANOVA (Análisis de Varianza)

Compara medias de tres o más grupos para determinar si existen diferencias significativas.

🎯 Aplicación en laboratorios:

• Comparar resultados entre operadores
• Evaluar diferencias entre equipos
• Analizar variabilidad entre lotes

F = Varianza entre grupos / Varianza dentro de grupos

F calculado > F crítico → Existen diferencias significativas

🔬 Prueba t de Student

Compara medias de dos grupos. Existen dos tipos principales:

📌 Tipos:

t pareada: Mismas muestras antes/después
t independiente: Dos grupos diferentes

t = (x̄₁ - x̄₂) / √(s₁²/n₁ + s₂²/n₂)

|t| > t crítico → Diferencia significativa (p < 0.05)

🔄 Repetibilidad y Reproducibilidad

Repetibilidad (r): Variabilidad bajo mismas condiciones (mismo operador, mismo equipo, corto plazo)

Reproducibilidad (R): Variabilidad bajo diferentes condiciones (diferentes operadores, equipos, laboratorios)

s_r = √(Σ(x_i - x̄)²/(n-1))

📏 Linealidad y Exactitud

Exactitud: Cercanía entre el valor medido y el valor verdadero

Linealidad: Capacidad de producir resultados proporcionales a la concentración

y = mx + b

Coeficiente de correlación R² > 0.995 → Linealidad aceptable

🏅 Ensayos de Aptitud (EA)

Evaluación externa de competencia técnica mediante comparación interlaboratorios.

📊 Indicadores clave:

Z-score: (x - X)/σ
|Z| ≤ 2 → Resultado satisfactorio
2 < |Z| < 3 → Cuestionable
|Z| ≥ 3 → No satisfactorio

📋 Conclusiones y Buenas Prácticas

✅ Buenas Prácticas en Estadística para Laboratorios

1. Verificar normalidad antes de aplicar pruebas paramétricas
2. Documentar todos los cálculos y procedimientos estadísticos
3. Utilizar gráficos de control para monitoreo continuo
4. Calcular incertidumbre considerando todas las fuentes de variabilidad
5. Participar regularmente en ensayos de aptitud

📌 Interpretación de Resultados

p-valor < 0.05: Diferencia estadísticamente significativa
R² > 0.995: Linealidad excelente
CV < 5%: Repetibilidad aceptable
|Z| ≤ 2: Desempeño satisfactorio en EA
